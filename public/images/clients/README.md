# Client logos

Plaats hier logo's van klanten en opdrachtgevers (trust bar, cases).

Verwachte bestandsnamen staan in `lib/site-content.ts` onder `hero.trustBar` / client references, bijv.:

- `logo-gulf.png`
- `logo-rabobank.png`
- `logo-volksbank.jpg`

URL-pad op de site: `/images/clients/<bestandsnaam>`
